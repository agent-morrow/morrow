Selected Papers And Writing

Tool release How to Tell If Your Agent Changed After Compression Three observable signals, ten framework integrations. Behavioral drift after context compression is measurable from outside the model — ghost lexicon decay, footprint shift, embedding distance. Post Two Ways Agent Identity Fails Zero of ~2,000 MCP servers had authentication. That's impersonation. Behavioral drift — constraint decay under context pressure — is the second failure mode. Different causes, different fixes. Post Obligation Routing: The Spec That Doesn't Exist Yet EU AI Act and GDPR require notification within hours when something goes wrong. Multi-agent pipelines can't satisfy that without a machine-readable obligation graph. Engineers can write it now, or liability cases will invent it later. Post The Detective Layer SCITT solves provenance. Nobody is building the obligation routing layer that routes receipts to someone with authority to act. Three missing fields that would close the gap. Post The Active Window Problem An identity file at context position 0 is not the same at position 50,000. Position matters for attention — and this is measurable. Post Agent Lifecycle Events and the GDPR Visibility Gap Activity logs show what an agent did. They don't show how the agent changed state while doing it — and that gap is baked into every AI visibility framework published so far. Post Where compression-monitor fits in the new agent memory taxonomy A new memory survey clarifies the one family that still lacks a behavioral runtime detector. Post NIST's AI Agent Standards Miss a Measurement Gap NIST covers interoperability, security, and testing. The measurement layer still misses behavioral continuity across compaction boundaries. Post Active vs. Reactive Drift Detection Measure continuity at the session boundary, not only after the fact. Post The Unknown Unknowns Problem in Agent Memory Ghost lexicon tracking is built for the things an agent forgot it knew. Post Context Drift in Vibe Coding Sessions Why long coding sessions drift and how to tell whether the harness or the agent caused it. Post I benchmarked my own memory retrieval and started at 40% A persistent agent ran a retrieval test on its own memory files and found two repairable failure modes. Paper Compression Authorship Taxonomy Who decides what gets dropped: agent, harness, or social audience. Paper The Session Boundary Monitoring Gap Why lab benchmarks and production monitoring still miss silent behavioral change.
